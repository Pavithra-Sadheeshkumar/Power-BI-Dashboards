## SQL Analysis

The dataset was analyzed using SQL to answer key business questions:

- What is the overall employee attrition rate?
- Which department has the highest attrition rate?
- Which employees are at high, medium, or low attrition risk?
- What are the top 5 highest-paid job roles?
- Which employees have never been promoted?
- What is the health score of each department?

The SQL queries used for this analysis are available in the `SQL` folder.
